Let's Play a Game


Directions: The point of the game is to move the pink square, so it collides with the black square that moves!

  1. Use keys I,J,K,L to move pink square
  2. If you catch the square the pink square will get smaller making it harder
  3. If you want some music press the play button below the playing field
  4. Challenge: See how many times you can catch the black square before the song is over!
